Acerca de mí

Inscrito en Garantía Juvenil con disponibilidad inmediata para incorporación remota o presencial.

Como programador, me gusta poner en primer lugar la lógica y la resolución de problemas, adaptándome a las necesidades del proyecto en cuanto a modelo de datos, patrones de diseño, arquitectura, renderizado en cliente o servidor etc. Consciente de la naturaleza colaborativa de la programación, mi prioridad es crear un código de calidad, limpio, optimizado y estructurado de acuerdo con las buenas prácticas que sea fácilmente mantenible.

Destrezas y aptitudes

  • Especialización en el framework Angular en Typescript con las librerías Angular Material y RxJS
  • Manejo de XMLHttpRequest, Axios, Fetch, HTTP Client (Angular) y HttpClient (Java)
  • Conocimiento avanzado de HTML, CSS (puro y con librerías como Bootstrap, Tailwind, Pico o Beer CSS), Javascript y Typescript
  • Formación en UX, UI, accesibilidad y diseño adaptado a ordenadores de sobremesa, laptops, tablets, Ipads y smartphones
  • Back-end con Java Spring Boot, NodeJS (Express) y PHP
  • Bases de datos relacionales SQL
  • Control de versiones con Git y GitHub
  • Inglés intermedio: B2 certificado por Cambridge Assessment English
  • Actitud activa y diligente para aprender y adaptarme
  • Trabajo en grupo con metodología Agile de Scrum
