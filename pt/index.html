Gerador de exames

Aplicação web Full Stack feita em Angular, Node.js e MySQL. Esta aplicação permite gerar exames aleatórios por tema e dificuldade, gerir as perguntas e respostas (CRUD) na base de dados, baixar para imprimir, resolver online e guardar no Local Storage os resultados obtidos

Lista de compras

PWA e APP (compilada com Capacitor) responsiva desenvolvida com Ionic e Angular que permite gerir a sua despensa de alimentos e gerar automaticamente a lista de compras. Esta aplicação utiliza Ionic Storage para armazenamento local persistente e um serviço com observables que transmite os dados dentro da aplicação utilizando RxJS

Biblioteca pessoal

Aplicação Single-Page (SPA) responsiva desenvolvida em React com JavaScript utilizando as bibliotecas Tailwind CSS e SheetJS. Esta aplicação permite gerenciar sua biblioteca com diferentes campos, filtrar os livros, salvá-los no Local Storage, exportar e importar os dados da aplicação em formato JSON e baixá-los em formato XLSX

Plano de treino

Responsive Single-Page Application (SPA) feita em TypeScript com o framework Angular e com as bibliotecas Angular Material, SheetJS e RxJS. Esta aplicação consome uma API própria de leitura apenas, usando RxJS para armazenar os dados num observable consumível em toda a aplicação de acordo com os princípios da programação reativa

Gestão de empregados

Aplicação web Full Stack com base de dados MySQL e início de sessão que permite gerir os seus próprios empregados, associados de forma única ao seu usuário. Feita com HTML, CSS, Javascript, PHP, MySQL e a biblioteca Bootstrap 5
